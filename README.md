# BugProwler

BugProwler is a repository containing several agents designed to help you analyze, identify, and document bugs more effectively. This document provides details on the agents included in the repository as well as their functions and setup instructions.

## Existing Agents

### Agno Assist

**Function:**
Agno Assist is designed to assist with automatic code reviews, providing suggestions and identifying potential issues in the codebase. It is equipped with an advanced knowledge graph to understand code dependencies and potential areas of improvement.

**Setup Instructions:**
1. Clone the repository:
   ```bash
   git clone https://github.com/ivangrana/bugProwler.git
   ```
2. Navigate to the repository root:
   ```bash
   cd bugProwler
   ```
3. Follow any installation guidelines in the `agents/agno_assist` directory for specific setup instructions.

### Recon Agent

**Function:**
Recon Agent specializes in scanning and gathering metadata from repositories. It finds key information, highlights patterns, and summarizes data insights, enabling structured exploration of the project's health and dependencies.

**Setup Instructions:**
1. Clone the repository:
   ```bash
   git clone https://github.com/ivangrana/bugProwler.git
   ```
2. Navigate to the repository root:
   ```bash
   cd bugProwler
   ```
3. Follow any installation guidelines in the `agents/recon_agent` directory for specific setup instructions.
